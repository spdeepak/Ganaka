## Features

- Clean, minimal UI
- Native macOS experience
- Lightning-fast calculations
- Lightweight and unobtrusive
- Multi-tab workspaces with drag-and-drop reordering
- Auto-save for `.ganaka` workspace files
- Undo/Redo support (Cmd+Z / Cmd+Shift+Z)

## Types of calculations

### Basic Arithmetic
- 2 + 3, 10 - 4, 5 * 6, 20 / 4
- Parentheses: (2 + 3) * 4
- Unary minus: -5
- Unicode operators: ×, ÷, −
### Math Functions
- abs(-5), sqrt(16), round(3.7), floor(3.7), ceil(3.2)
- sin(1), cos(1), tan(1), log(100), ln(e), exp(1)
### Constants
- pi / π, e
### Line References
- `$1`, `$2`, `$last` — reference results from other lines
- `prev` — previous line's result
### Aggregates
- `sum` / `total`, `avg` / `average` / `mean`, `count`, `min`, `max`
### Random
- `random` / `rand` — random number 0–1
### Percentages
- 20% of 500 → 100
- 50 is what % of 200 → 25%
- 100 + 10% → 110
- 100 - 10% → 90
- 50% → 0.5
### Variables
- tax = 0.08 — assign, then use tax * 1000
### Currency Conversion
- 100 USD to EUR, 50 GBP to INR
- Live exchange rates via CurrencyRateService
### Unit Conversions
- Length: 10 km to miles, 5 ft to cm, etc.
- Weight: 100 kg to lb, 16 oz to g, etc.
- Volume: 1 gal to l, 500 ml to fl oz, etc.
- Time: 2 h to min, 3600 s to h, etc.
- Temperature: 100 F to C, 0 C to K, 32 F to K, etc.
### Date Calculations
- today, tomorrow, yesterday
- today + 90 days, today - 2 weeks, today + 3 months
- June 27, Jan 1 2025, Dec 25, 2026
- June 27 + 90 days, Jan 1 - 30 days, Dec 25 + 2 weeks
### Special
- `#` prefix — comment lines (ignored in calculations)
